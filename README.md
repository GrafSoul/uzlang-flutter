## 💡 О проекте

**UzLang** — мобильное приложение для изучения узбекского языка не-узбеками (в первую очередь русскоязычными). Формат — карточки с геймификацией: учишь слова блоками, открываешь фразы, держишь серию дней.

Это полноценная мобильная реинкарнация веб-версии [GrafSoul/uzlang](https://github.com/GrafSoul/uzlang) — с новым дизайном, новой архитектурой и интервальным повторением.

**Всё работает офлайн.** Контент, прогресс и алгоритм повторения живут локально на устройстве — сервер не нужен.

## 📚 Педагогическая модель

Ядро методики — **сначала слова, потом фразы из этих слов**:

1. **Слова блоками по 20**: Учить → Повтор → Тест (нужно 10/10) → блок закрыт.
2. **Фразы открываются по теме** — когда выучены *все* слова темы. Запертая тема показывает прогресс: `Базар (5/20)`.
3. **Фразы читаются легко** — они собраны из уже знакомых слов и закрепляют их в контексте.
4. **FSRS** (современный алгоритм интервального повторения) планирует due-повторы и слов, и фраз.

## ✨ Возможности

- 🃏 **Три режима**: Учить (свайп-карточки), Повтор (FSRS), Тест (выбор варианта, сборка фразы)
- 🔥 **Геймификация**: XP, серия дней (streak), дневная цель, недельная активность
- ❤️ **Жизни**: дневной пул из 3, восстановление в полночь с обратным отсчётом
- 📊 **Прогресс**: выучено, точность, реальная история активности по дням
- 🔊 **Озвучка**: системный TTS с узбекским; если голоса нет — турецкий фонетический фолбэк
- ✍️ **Письменность на выбор**: латиница / кириллица / обе — применяется на лету
- 🌗 **Тёмная и светлая темы** с мгновенным переключением
- 🔔 **Ежедневное напоминание** о занятии
- 📴 **100 % офлайн** — без регистрации, без сервера
- ▶️ **Resume**: урок продолжается с того места, где остановился

## 🏗 Архитектура

Clean-ish, feature-based. Контроллеры тонкие, бизнес-логика в domain-сервисах. UI никогда не ходит в БД напрямую — только через репозитории.

```
lib/
├── app/            # роутинг, DI-биндинги (InitialBinding)
├── core/           # тема (DS из Figma), сервисы, константы, утилиты
├── domain/         # сущности (Freezed), интерфейсы репозиториев, сервисы
│                   #   (LearningService, FsrsScheduler, Gamification, TopicProgress…)
├── data/           # Drift: схема, DAO, сидер; реализации репозиториев
├── features/       # экраны: onboarding, home, topics, topic_detail,
│                   #   learn, phrases, result, progress, profile, shared
└── l10n/
```

В V1 заложены **архитектурные швы** под будущее без переписывания: авторизация (`UserService` → пока локальный анонимный профиль), подписка (`AccessService` → пока всегда premium), облачный синк и контент (репозитории → пока только локально).

Подробнее: [docs/architecture.md](docs/architecture.md) · [docs/implementation-plan.md](docs/implementation-plan.md)

## 🎨 Дизайн

Премиум-тёмный стиль, акцент `#FF8A3D`, шрифт Inter, единые SVG line-иконки. Все экраны реализуются **1:1 по макетам Figma** — дизайн-система (цвета, типографика, радиусы) выгружена из Figma-переменных, не «на глаз».

## 🚀 Запуск

Требуется Flutter **3.44+** (Dart 3.8+).

```bash
# Зависимости
flutter pub get

# Кодогенерация (Drift, Freezed) — сгенерированные файлы не в git
dart run build_runner build --delete-conflicting-outputs

# Запуск
flutter run
```

При первом старте приложение автоматически сеет контент из `assets/seed/content.json` в локальную БД.

flutter analyze   # zero issues
